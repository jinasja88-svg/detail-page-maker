# 개발 진행 현황

## 완료된 기능

### 1번: 크롤링 (crawler.py)
- 쿠팡/네이버/카페24 자동 감지
- Playwright + Stealth로 브라우저 크롤링
- 캡차 감지 → 사용자 입력 → 자동 통과 (on_captcha 콜백)
- 네이버: data-src lazy-load 처리, 펼쳐보기 클릭 후 스크롤
- 쿠팡: 더보기 클릭 후 스크롤
- 프록시 지원

### 2번: 이미지 분할 (image_splitter.py)
- AI 없이 Pillow + numpy로 여백 감지 분할
- 세로/가로 비율 3:1 이상만 분할
- 여백 30px 이상, 섹션 최소 500px
- split_all_images()로 폴더 전체 일괄 처리

### 3번: 섹션 분석 + HTML 재현 (테스트 완료)
- Claude 자체 비전으로 섹션별 디자인 요소 추출 (배경, 텍스트 위치/크기/색상, 제품 위치)
- GPT-4o도 테스트 완료 (852토큰/섹션, 약 9원)
- HTML/CSS로 원본 컨셉 재현 + contenteditable로 텍스트 수정 가능
- DALL-E 3으로 제품 이미지 생성 ($0.04/장, 약 58원)
- 로지텍/바이젤디 두 상품으로 테스트 성공

### 진행 중: 상세페이지 리디자인 시스템
- 문구: 참조 홈페이지 1에서 추출
- 디자인/구성: 참조 홈페이지 2에서 추출
- 두 개를 조합하여 새 상세페이지 생성
- CRO 카피라이터 프롬프트 작성 완료 (14섹션 구조)

## 기술 결정사항
- 이미지 분할: AI 불필요 → Pillow 여백 감지로 충분
- 섹션 분석: Claude 비전이 GPT-4o보다 디테일 정확 (그라데이션, 반전박스, 아이콘 등)
- HTML 출력: contenteditable로 텍스트 실시간 수정 가능
- 이미지: DALL-E 3 Standard 1024x1024 ($0.04/장)
- 상세페이지 1개 전체 비용: 약 650~1,000원
