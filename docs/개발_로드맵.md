# 상세페이지 메이커 - 개발 로드맵

1. **크롤링** - 홈페이지 주소를 넣으면 기존 상세페이지를 크롤링
2. **페이지 분할** - 상세페이지를 페이지별로 분할 (AI 사용)
3. **콘텐츠 분석** - 상세페이지 내용을 텍스트와 그림으로 분리
4. **구성 파악** - 기존 상세페이지의 구성을 그림 하나씩 찾기
5. **내 제품 사진 삽입** - 내가 넣어주는 사진(내 제품)을 해당 상세페이지 참고해서 삽입
6. **구성안 재설계** - 기획 구성은 처음부터 다시 짜기 (페이지 하나 변경까지)
7. **AI 이미지 생성** - 해당 상세페이지를 이미지 생성 + 내 제품으로 삽입
8. **문구 추천** - 해당 제품의 장점을 살려서 어떤 문구를 넣으면 좋을지 찾기
9. **문구 배치** - 기존 제품 상세페이지를 참고해서 문구 배치 위치 결정 (AI)
10. **디자인 요소 적용** - AI 이미지에 텍스트를 기존 디자인 요소만 살려서 삽입 (기획순서 변경, 내 제품 사용)
11. **텍스트 분리** - 텍스트는 이미지와 개별로 넣을 수 있도록 분리 (편집 가능)
